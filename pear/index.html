Ryan Chouinard's PEAR channel

Registering the channel: 
pear channel-discover rchouinard.github.com/pear
Listing available packages: 
pear remote-list -c rych
Installing a package: 
pear install rych/package_name
Installing a specific version/stability: 
pear install rych/package_name-1.0.0
pear install rych/package_name-beta
Receiving updates via a feed: 
http://rchouinard.github.com/pear/feed.xml
